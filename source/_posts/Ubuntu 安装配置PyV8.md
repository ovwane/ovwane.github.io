# Ubuntu 安装配置PyV8

> Ubuntu 18.04.1 LTS

**安装依赖：** 
Boost

```shell
apt install libboost-dev
```

安装v8

```shell
brew install v8
```

安装PyV8

```shell
pip install pyv8
```
