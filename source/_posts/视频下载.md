下载Youtube视频

https://github.com/soimort/you-get

https://github.com/rg3/youtube-dl

安装youtube-dl

```shell
brew install youtube-dl
```

下载

```shell
# 获取所有视频格式id
youtube-dl -F URL

# 下载视频
youtube-dl -f id URL
```
