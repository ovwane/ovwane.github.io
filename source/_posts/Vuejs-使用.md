**安装Vue CLI 3**

```shell
npm install -g @vue/cli
npm install -g @vue/cli-service-global
```

**查看版本**

```shell
vue --version
```

## 本地预览

### serve

```shell
npm install -g serve
```

**运行serve**

```shell
serve -s FrontEndGuide
```

**访问**

http://localhost:5000
